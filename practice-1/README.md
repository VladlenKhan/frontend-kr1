# Практика №1 — CSS-препроцессоры (SASS)

**Тема:** Магазин кроссовок

**Использованные возможности SASS:**
- Переменные (`$primary`, `$dark-bg` и др.)
- Миксины (`@mixin button`)
- Вложенность селекторов
- Условные конструкции (`@if $theme == dark`)
- Модификаторы (`sneaker-card--accent`)

| Возможность SASS              | Где использована                          | Конкретный код / место                          | Зачем использована (соответствие методичке) |
|-------------------------------|-------------------------------------------|--------------------------------------------------|---------------------------------------------|
| **Переменные**                | `styles.scss` (строки 4–9)               | `$primary`, `$dark-bg`, `$light-bg`             | Однократное определение цветов и размеров для удобства изменения |
| **Миксины**                   | `styles.scss` (строки 12–20)             | `@mixin button($color)`                         | Повторное использование блока стилей кнопки |
| **Вложенность селекторов**    | `styles.scss` (строки 35–70)             | `.sneaker-card { ... &__title { ... } }`       | Повторяет структуру HTML, уменьшает повторения |
| **Условные конструкции**      | `styles.scss` (строки 72–76)             | `@if $theme == dark { ... }`                    | Переключение между светлой и тёмной темой |
| **Модификаторы**              | `styles.scss` (строки 55–58)             | `&--accent { border: 3px solid $primary; }`    | Добавление акцентной рамки к карточке |
| **Математические операции**   | `styles.scss` (строки 18, 42)            | `padding: 12px 24px`                            | Удобство работы с размерами |

**Файлы практики:**
- `index.html` — семантическая разметка карточек
- `styles.scss` — весь препроцессорный код
- `styles.css` — скомпилированный результат

**Как запустить:**
1. Открыть `index.html` в браузере
2. (Опционально) `sass --watch styles.scss:styles.css`
