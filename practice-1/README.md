# Практика №1 — CSS-препроцессоры (SASS)

**Тема:** Магазин кроссовок

**Использованные возможности SASS:**
- Переменные (`$primary`, `$dark-bg` и др.)
- Миксины (`@mixin button`)
- Вложенность селекторов
- Условные конструкции (`@if $theme == dark`)
- Модификаторы (`sneaker-card--accent`)

**Как запустить:**
1. Открыть `index.html` в браузере
2. (Опционально) `sass --watch styles.scss:styles.css`
