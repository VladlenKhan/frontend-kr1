# Практика №5 — Расширенный REST API + Swagger

**Тема:** Магазин кроссовок

**Реализовано:**
- Полная документация Swagger (/api-docs)
- JSDoc-комментарии для всех маршрутов
- Схема Sneaker
- Интерактивное тестирование прямо в браузере
- `swagger-jsdoc` + `swagger-ui-express`
- JSDoc-комментарии для всех маршрутов
- Схема `Sneaker` в `components/schemas`
- Автоматическая генерация интерактивной документации

**Где применено:**
- `app.js` — все `@swagger` комментарии
- Маршруты `/sneakers` и `/sneakers/{id}` полностью задокументированы

**Результат:** Документация доступна по адресу `/api-docs`.

**Как запустить:**
cd backend
npm start

Открыть в браузере: http://localhost:3001/api-docs
