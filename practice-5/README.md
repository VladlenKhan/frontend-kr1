# Практика №5 — Расширенный REST API + Swagger

**Тема:** Магазин кроссовок

**Реализовано:**
- Полная документация Swagger (/api-docs)
- JSDoc-комментарии для всех маршрутов
- Схема Sneaker
- Интерактивное тестирование прямо в браузере

**Как запустить:**
cd backend
npm start

Открыть в браузере: http://localhost:3001/api-docs
