# Практика №2 — Сервер на Node.js + Express

**Тема:** Магазин кроссовок (CRUD API)

**Реализовано:**
- GET /sneakers — список всех товаров
- GET /sneakers/:id — один товар
- POST /sneakers — добавление
- PATCH /sneakers/:id — обновление
- DELETE /sneakers/:id — удаление


### Использованный код и его назначение

| Конструкция                        | Файл / строки               | Для чего использована |
|------------------------------------|-----------------------------|-----------------------|
| `express.json()`                   | `app.js` (строка 8)         | Парсинг JSON-тела запроса |
| `app.get('/sneakers')`             | `app.js` (строки 28–30)     | Получение всех товаров |
| `app.get('/sneakers/:id')`         | `app.js` (строки 32–37)     | Получение товара по ID |
| `app.post('/sneakers')`            | `app.js` (строки 39–48)     | Создание нового товара |
| `app.patch('/sneakers/:id')`       | `app.js` (строки 50–57)     | Частичное обновление |
| `app.delete('/sneakers/:id')`      | `app.js` (строки 59–63)     | Удаление товара |
| `res.status(201)`, `res.status(404)` | `app.js`                   | Правильные HTTP-статусы |


Все 5 CRUD-операций реализованы согласно заданию практики №2. Использован middleware `express.json()` для работы с телом запроса.

**Как запустить:**

```bash
# 1. Установка зависимостей
npm install

# 2. Запуск сервера
npm start
