## Что реализовано

Полноценное веб-приложение «Магазин кроссовок» с полным CRUD.

### Использованные технологии и практики:
- **Практика 1** → SASS (переменные, миксины, вложенность, условная тёмная тема) — [перейти к practice-1](../practice-1)
- **Практика 2** → Express.js + базовый CRUD — [перейти к practice-2](../practice-2)
- **Практика 3** → Postman тестирование — [перейти к practice-3](../practice-3)
- **Практика 4** → React + fetch + соединение фронтенда и бэкенда — [перейти к practice-4](../practice-4)
- **Практика 5** → Swagger документация — [перейти к practice-5](../practice-5)

### Функционал финального проекта:
- Просмотр всех кроссовок
- Добавление, редактирование, удаление
- Красивые карточки на SASS
- Интерактивная документация Swagger (`/api-docs`)
- CORS, валидация, обработка ошибок

## Как запустить проект

```bash
# 1. Backend
cd backend
npm install
npm start
# → запустится на http://localhost:3001 + Swagger на /api-docs

# 2. В новом терминале — Frontend
cd frontend
npm install
npm start
# → откроется http://localhost:3000
