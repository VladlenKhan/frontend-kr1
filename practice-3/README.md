# Практика №3 — JSON и внешние API + Postman

**Тема:** Магазин кроссовок

**Что сделано:**
- Используется API из Практики 2 (`http://localhost:3000/sneakers`)
- Создана коллекция в Postman со всеми CRUD-запросами
- Протестированы: GET, POST, GET/:id, PATCH, DELETE
- JSON-ответы соответствуют структуре из занятия

**Как проверить:**
1. Запустить сервер из `practice-2` (`npm start`)
2. Открыть Postman → Import → выбрать файл `Sneaker-Store-Postman-Collection.json`
3. Запустить запросы по порядку
4. Скриншоты результатов лежат в папке `screenshots/`
